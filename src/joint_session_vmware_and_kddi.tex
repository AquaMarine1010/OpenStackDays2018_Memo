VIOの主な特徴

\begin{itemize}
\item 簡便なインストール
\item アップグレードが簡便
\item インストール・アップグレードでバグを踏まない
\item ライブマイグレーション・HA・QoS・LBに対応
\item vCenterからの操作可能
\item 1バージョン飛しでのアップグレードが可能
\end{itemize}
